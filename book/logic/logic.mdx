# Logic

<Box type={"Axiom"} name={"Axioms of Equality"}>

**Axiom 1 (Reflexive).** For every <M math="x \in \mathbb{N}, x = x"/>.

**Axiom 2 (Symmetry).** For every <M math="x, y \in \mathbb{N}"/>, if <M math="x = y" />, then <M math="y = x"/>.

**Axiom 3 (Transitive).** For every <M math="x, y, z \in \mathbb{N}"/>, if <M math="x = y"/> and <M math="y = z"/>, then <M math="x = z"/>.

**Axiom 4 (Closed).** For all <M>x</M> and <M>y</M>, if <M math="x \in \mathbb{N}"/> and <M math="x = y" />, then <M math="y \in \mathbb{N}"/>.

</Box>

The Peano axioms serve to formally define the natural numbers, refered to as the set <M math="\mathbb{N}"/>.

**Axiom 1 (Zero).** <M math="0"/> is a natural number. That is, <M math="0 \in \mathbb{N}"/>.

**Axiom 2 (<M math="S"/> is closed).** If <M math="x \in \mathbb{N}" />, then <M math="S(x) \in \mathbb{N}"/>&nbsp;.

**Axiom 3.** For every natural number <M math="x \in \mathbb{N}"/>, <M math="S(x) = 0"/>&nbsp; is false (i.e. <M math="0"/> is not the successor of any natural number).

**Axiom 4 (Injective).** For all <M math="x, y \in \mathbb{N}"/>&nbsp;, if <M math="S(x) = S(y)"/>&nbsp;, then <M math="x = y"/>.
